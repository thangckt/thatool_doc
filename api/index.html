    Minimization is a process of finding the minimum energy configuration of the atoms. This take a lot of time and resources. The idea of this function is to perform a rough minimization, using a cheap method, e.g., ... and less accuracy parameters (e.g., force/enegy convergence criterion), before performing the actual minimization with a more accuracy and expensive method. See this discussion

    Optimize the structure using the Effective Medium Potential (EMT) calculator. This is a very simple calculator, which is not very accurate, but it is very fast. It is useful for a rough optimization before using a more accurate calculator. See this discussion

    Supports the following standard EMT metals: Al, Cu, Ag, Au, Ni, Pd and Pt. In addition, the following elements are supported. They are NOT well described by EMT, and the parameters are not for any serious use: H, C, N, O. All supported potentials in EMT.
